# 3D Shooter Prototype

A small FPS-style prototype built with Vite + TypeScript + Three.js.

## Controls
- `WASD` — move
- Mouse — look
- Left click — shoot
- `ESC` — release pointer lock
- Click **"Click to Start"** — enables pointer lock + starts playing

## Getting Started
Prereqs: Node.js (recommend 18+)

Install deps:
```bash
npm install
```

Run dev server:
```bash
npm run dev
```

Build:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Notes
- Prototype / experiment project.
- Build output is in `dist/` (ignored by git).
